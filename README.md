# 한줄광고 모듈 (PlusAd)

포인트를 사용해 한줄 광고를 등록하고 이를 위젯으로 출력하는 [라이믹스(Rhymix)](https://github.com/rhymix/rhymix) 모듈입니다.

![미리보기](preview.png)

---

## 주요 기능

- 회원이 보유한 포인트를 소모하여 한줄 광고를 등록할 수 있습니다.
- 등록된 광고는 위젯을 통해 원하는 위치에 출력됩니다.
- 광고 진행 시간을 시간(hour) 단위로 설정할 수 있습니다.
- 광고 텍스트에 **굵기(Bold) 효과** 및 **색상 효과**를 적용할 수 있습니다 (추가 포인트 소모).
- 광고에 외부 링크(URL)를 첨부할 수 있습니다 (화이트리스트 방식으로 허용 도메인 제한).
- 내 광고 목록 및 광고 수정·삭제 기능을 제공합니다.
- 관리자 페이지에서 광고 로그 확인 및 삭제, 권한 설정, 스킨 설정이 가능합니다.

---

## 요구 사항

- [라이믹스(Rhymix)](https://github.com/rhymix/rhymix) CMS
- **[한줄광고 위젯](https://github.com/Lastorder-DC/rx-widget-plusad)** (실제 광고 출력을 위해 별도 설치 필요)

---

## 설치 방법

1. 이 저장소를 클론하거나 압축 파일을 다운로드합니다.
2. 폴더명이 `plusad`인지 확인한 뒤, 라이믹스가 설치된 경로의 `modules/` 디렉터리 아래에 복사합니다.
   ```
   (라이믹스 루트)/modules/plusad/
   ```
3. 관리자 대시보드에서 DB 생성 및 업데이트를 진행합니다.
4. **[한줄광고 위젯](https://github.com/Lastorder-DC/rx-widget-plusad)**을 별도로 설치한 뒤, 위젯을 원하는 위치에 추가합니다.

---

## 관리자 설정

관리자 페이지 → 설치된 모듈 → **한줄광고 모듈** 메뉴에서 아래 항목을 설정할 수 있습니다.

| 설정 항목 | 설명 | 기본값 |
|---|---|---|
| 광고 진행시간 | 허용할 광고 진행 시간 목록 (시간 단위, 쉼표로 구분) | `1,3,6,12,24` |
| 시간당 소모포인트 | 광고 등록 시 시간당 차감할 포인트 | `10` |
| 굵기효과 소모포인트 | Bold 효과 적용 시 추가로 차감할 포인트 | `100` |
| 색상효과 소모포인트 | 색상 효과 적용 시 추가로 차감할 포인트 | `300` |

---

## 광고 등록 방법 (사용자)

1. 광고 등록 권한이 있는 회원으로 로그인합니다.
2. 한줄광고 모듈 페이지에서 **광고 등록** 버튼을 클릭합니다.
3. 광고 내용, 링크 주소(선택), 진행 시간, 굵기·색상 효과를 선택합니다.
4. 소모될 포인트가 충분한 경우 광고가 등록됩니다.
