# Changelog

모든 주요 변경 사항은 이 파일에 기록됩니다.  
버전은 `conf/info.xml`의 버전 정보 변경을 기준으로 구분합니다.

---

## [1.2.0] - 2026-04-10

### 추가
- 포인트 모듈 비활성화 여부 확인 기능 추가 (#5)
  - 포인트 모듈이 비활성화된 경우 포인트 잔액 무제한으로 처리
- 광고 등록 가능 시간 유효성 검사 동적 처리 추가 (#5)
- `dispPlusadCancel` 및 `matchWhitelist` 함수에 대한 PHPUnit 테스트 추가 (#4)
- GitHub Actions CI 파이프라인 추가 (PHP 7.4~8.5 매트릭스 테스트) (#4)

### 변경
- 경과 시간(elapsed percent) 계산 로직 및 조건 리팩터링
- 시간 표시 부분 최적화
- `$allowed_times`가 빈 값이 될 수 없도록 처리 수정
- 포인트 모듈 설정 확인 시 `Point::getConfig()` 사용으로 코드 개선

### 수정
- 취소 기능 관련 누락된 부분 추가 수정

## [1.0] - 2026-02-21

### 추가
- GNU Lesser General Public License v2.1 적용

- 설치 방법 및 관리자 설정 안내 문서 수정
- 라이선스 GPL로 변경 (Rhymix 모듈 규정 준수)
- 위젯 설치 안내 포맷 수정
- 광고 외부 링크 첨부 방식 설명 명확화
- `conf/module.xml` 코드 정리 (불필요한 주석 및 잔여 코드 제거)
- `plusad.controller.php` 리팩터링
